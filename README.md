# 整理一个适合快速开发的版本

## 版本基于 DedeCMS-V5.7-UTF8-SP2 (2017-04-05)

### 版本特性
- 通用的后台登陆页
- 通用的后台主页
- 移动端管理发布功能

### 安全备注
- 如果没有会员系统，删除 member 文件夹
- 安装后删除 install 文件夹
- 如果没有专题系统，删除 special 文件夹
- 如果没有问答模块，删除 ask 文件夹

### 移动端备注
- PC端模板必须以 `.htm` 结尾，移动端模板需与PC端名字相同，以 `_m.htm` 结尾
- 移动端框架采用 Amaze UI
- 移动端文件生成目录在 /m/ 文件夹中
